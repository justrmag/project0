Game Aesthetics
-basically these are the rules or objectives of the game.
-These are how the players use the set rules of the game.
-This is how the game looks, feels and how it makes the player feel.
Game Mechanics are the first thing that a game designer looks at when preparing and planning to design. This job is mainly done by the software developers or coders of the game.
This incorporates both the coders and the artists of the game to implement the mechanics in the right way.
Mostly for artists, storytellers and less of coders. This is the first thing the player sees when they play the game thus making the 3rd element extremely crucial.
